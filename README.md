# HR_Analytics_Dashboard – Employee Attrition Analysis

This project focuses on analyzing employee attrition and workforce trends using Power BI. The dashboard provides key HR insights such as employee count, attrition rate, department-wise attrition, salary-based attrition, job satisfaction analysis, age distribution, gender distribution, and experience trends.

The goal of this project is to help HR teams and business leaders identify the major factors contributing to employee turnover, monitor workforce demographics, and make data-driven decisions to improve employee retention and organizational performance.

Key Insights
Total Employees, Active Employees, and Attrition Count
Attrition Rate Analysis
Department-wise Attrition Breakdown
Salary-wise Attrition Analysis
Job Role & Job Satisfaction Insights
Age Group and Gender Distribution
Experience and Tenure Analysis
Interactive Filters for Detailed Exploration
Tools & Technologies
Power BI
Excel / CSV Dataset
Data Cleaning & Transformation
DAX Measures
Interactive Dashboard Design
Business Impact
Identify departments with high attrition
Understand employee satisfaction patterns
Support workforce planning and retention strategies
Improve HR decision-making through data visualization

This dashboard transforms raw HR data into actionable insights, enabling organizations to better understand employee behavior and reduce attrition.
